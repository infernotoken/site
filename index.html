How It Works

1. There was only 1,000,000 Inferno ever created.


2. There will never be any more Inferno minted.

3. Every time there is a Inferno transaction, 1% of the transacted amount will be burnt.



The intention of Inferno is not to be used for regular day-to-day transactions, but rather be intended to be used as a decentralized hedge against traditional inflationary instruments.
